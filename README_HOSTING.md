# Turbo Hosting System

## المجلدات المهمة
- logs: جميع اللوقات (أوامر، أمان، حالة السيرفر)
- scripts: أدوات مساعدة (فحص صحة، عارض لوقات، اختبار حماية)
- servers: ملفات السيرفرات

## ميزات الأمان
- حماية من السبام
- حماية من الأوامر الضارة
- لوقات مفصلة
- توليد صور إحصائية

## أدوات سريعة
- scripts/health_check.php: فحص صحة السيرفر
- scripts/log_viewer.php?file=commands.log: عرض اللوقات
- scripts/security_test.php: اختبار حماية الأوامر
